# Youth and Development Network in East Africa (YDNEA)

"Dare to Dream, Dare to Achieve."

## Overview
A world-class, fully responsive website and community management platform for Youth and Development Network in East Africa (YDNEA). Built with Flutter, this platform features a modern international NGO/community design with a Light Blue and White premium UI.

## Features
- **Responsive Design**: Works on mobile, tablet, and desktop.
- **Home Screen**: Hero section with CTAs to empower young people.
- **Learn More**: Detailed information about the organization, mission, vision, and core values.
- **Leadership Platform**: Registration forms and health insurance portals.
- **Theming**: Premium glassmorphism and modern NGO aesthetic.

## Tech Stack
- Flutter (Web, iOS, Android, Desktop)
- Dart

## Setup
1. Run `flutter pub get`
2. Run `flutter run -d chrome` to preview the web app.
